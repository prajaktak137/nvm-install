NVM Setup Guide
STEP 1
Install NVM from https://github.com/coreybutler/nvm-windows/releases
Download executable setup file, click on the file named nvm-setup.exe and finish the setup.
STEP 2
Once install check via "nvm -v"
STEP 3
Install your desire npm version by visiting https://nodejs.org/en/download/releases/
STEP 4
Copy version name from node website. From cmd "nvm install v12.11.1"
STEP 5
Check installed all version via "nvm list"
STEP 6
Change node version via "nvm use 12.11.1"
